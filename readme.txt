== Installation ==

1. Upload `lunchcomWidgetManager.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on "Appearance" and then "Widgets" and then drag the Lunch Communities widget into the appropriate sidebar
4. Enter the url for your community.  For example, for http://www.lunch.com/MBLocal, enter <b>MBLocal</b>
5. Select the widget type you wish to display.
